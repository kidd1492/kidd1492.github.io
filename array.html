    JavaScript Array Methods

    Iteration Methods

    • forEach(): Iterates over each element and executes a provided function.
    • map(): Creates a new array by applying a function to each element.
    • filter(): Creates a new array with elements that pass a specified test.
    • reduce(): Reduces an array to a single value by applying an accumulator function.
    • every(): Checks if all elements satisfy a condition.
    • some(): Checks if at least one element satisfies a condition.

    Transformation Methods

    • map(): Transforms each element and creates a new array.
    • sort(): Sorts elements in the array (in-place or by creating a new array).
    • reverse(): Reverses the order of elements in the array.
    • concat(): Combines two or more arrays into a new array.
    • slice(): Creates a shallow copy of a portion of the array.
    • splice(): Adds or removes elements from the array.

    Utility Methods

    • join(): Joins array elements into a string.
    • toString(): Converts the array to a string.
    • isArray(): Checks if an object is an array.
    • length: Property that returns the number of elements in the array.
