auto_django

auto_django Sets up a new Django project with a virtual environment and installs Django. Generates a new app with necessary directories for templates and static files. Creates essential starter files like style.css, layout.html, and index.html. Configures URL routing for the app. Adds custom templates for different views.Runs makemigrations and migrate to create the database. Just createsuperuser and set up the account. when you log into the admin area the tables will be registered.
